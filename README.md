👋 About Me

I'm a developer who builds complete products for startups — mobile apps, backend systems, and automation, all working together as one system. With Flutter and FastAPI as my core stack, I take a product from idea to a working release on my own.

Lumenbit is one of my startups — a studio where I design and build mobile apps, web apps, and advanced Telegram bots for early-stage businesses.

<br/>

من توسعه‌دهنده‌ای هستم که محصولات کامل رو برای استارتاپ‌ها می‌سازم — اپلیکیشن موبایل، سیستم بک‌اند و اتوماسیون، همه در قالب یک سیستم یکپارچه. با تکیه بر Flutter و FastAPI به‌عنوان استک اصلی، یک ایده رو به‌تنهایی تا نسخه‌ی نهایی و قابل استفاده می‌رسونم.
